# 🎓 Career Guidance Assessment System

A web-based career guidance and self-assessment platform designed for **Class 8 & 9 students**.  
The system helps students understand their **study habits, learning styles, and aptitude**, and generates a consolidated report after completing all assessments.

---

## 🚀 Project Overview

Choosing the right career path at an early stage can be confusing for both students and parents.  
This project provides a **structured, test-based approach** to help identify:

- Study behavior patterns  
- Preferred learning styles  
- Basic aptitude strengths  

The results can later be used by **career counsellors** to guide students effectively.

---

## 🧠 Assessments Included

1. **Study Habits Assessment**
   - Time management
   - Concentration
   - Revision habits
   - Exam preparedness

2. **Learning Style Assessment**
   - Visual learning
   - Auditory learning
   - Kinesthetic learning

3. **Aptitude Assessment**
   - Numerical ability
   - Verbal reasoning
   - Logical thinking

Each assessment is:
- Timed (10 minutes)
- Single-attempt only
- Auto-submitted if time expires

---

## 🔄 Application Flow

1. User selects **grade**
2. User chooses and completes:
   - Study Habits Test
   - Learning Style Test
   - Aptitude Test
3. ✔ A tick mark appears once a test is completed
4. 🔒 Completed tests cannot be retaken
5. 📄 **Report download is enabled only after all 3 tests are completed**

---

## ✨ Key Features

- Clean and responsive UI
- Progress tracking using `localStorage`
- Tick indicators for completed tests
- Disabled re-attempts for assessments
- Centralized report unlock logic
- No backend required (pure frontend)
